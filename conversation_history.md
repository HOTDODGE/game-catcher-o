# 디자인 시스템 및 테스트 사이트 구축 대화 기록

## 1. 초기 요청 및 확인
* **사용자**: "디자인 시스템 만든 거 있어?"
* **AI (Antigravity)**: 이전 대화 기록을 확인해 본 결과, CSS 변수를 활용하여 웹 디자인용 기본 디자인 시스템(디자인 토큰)을 기획하신 내용이 확인되었습니다. 당시 중점적으로 다루고자 했던 기초 디자인 요소들은 Color(색상), Typography(서체/타이포그래피 설정), Spacing(여백 및 간격), Border Radius(테두리 둥글기), Shadow(그림자 효과)였습니다. 당시 작성된 CSS 코드가 파일 형태로 보관되어 있지는 않아서, 제가 새롭게 구성해서 작성해 드릴 수 있다고 안내해 드렸습니다.

## 2. 디자인 시스템 및 테스트 사이트 구축 요청
* **사용자**: "이 디자인 시스템을 가지고 테스트 코드로 사이트를 만들어보자"
* **AI (Antigravity)**: 요청에 따라 CSS 파일(`style.css`)과 이를 테스트할 수 있는 HTML 파일(`index.html`)을 작성했습니다.
  * **`style.css`**: CSS 변수(:root)를 통해 Color, Typography(Inter 폰트), Spacing, Border Radius, Shadow 토큰을 정의했습니다. 또한 다크모드(`@media (prefers-color-scheme: dark)`) 지원과 Button, Card, Input 요소들의 기본 스타일을 구성했습니다.
  * **`index.html`**: 정의된 디자인 토큰들이 어떻게 시각적으로 나타나는지 확인할 수 있는 "테스트 랩" 페이지를 마크업했습니다. 색상 팔레트, 텍스트 계층, 버턴, 입력 폼, 카드형 요소들을 배치하여 실제로 작동하는 모습을 볼 수 있도록 구현했습니다.
